ðŸ”§ How to Use

Integrate the widget into your own projects

Installation

npm install html-ai-chat-widget

Basic Usage

<script src="https://cdn.jsdelivr.net/npm/html-ai-chat-widget"></script>
<script>
  HtmlAiChatWidget.setup({
    pages: ["/about.html", "/blog.html"],
    title: "Ask me anything",
    subtitle: "AI Assistant"
  });
</script>

Configuration Options

  • pages: Array of URLs to analyze for content
  • selector: CSS selector for the main content area (default: 'main')
  • maxSentences: Number of sentences in response (default: 2)
  • title: Widget title shown at the top
  • welcome: Welcome message when widget opens
  • analytics: Enable analytics tracking (default: false)
